Web Development - Missouri Kids

I have always been passionate about web development, so getting a chance to redesign a non-profit website from the ground up was an extremely enjoyable experience. I worked with a team of seven other students to change a very dated looking website into something completely different. I personally helped design the CSS and home page, as well as helped many other students with their tasks throughout the span of the project. This website was designed entirely in WordPress using the Avada theme. We coordinated with the website owner to design it to her specifications.

Web Development - Heat Up St. Louis

Heat Up St. Louis was a website I redesigned for a WordPress class, though this site never was pushed to live production I feel liked I learned the ins and outs of WordPress which I applied in its entirety to the Missouri Kids website. I began redesigning this website by planning it out with multiple wireframes. I also wanted to capture the concept of heating up St. Louis by capturing the skyline in blue and red. I decided to keep the layout as minimal as possible while focusing on getting people involved.

Javascript

This is a project I designed for my JavaScript final, I wanted to test my knowledge of Javascript and worked to design something that used JQuery and Ajax calls to pull data from an external source onto my page. The page uses JavaScript to populate the page with Pokemon based on searching by ID and/or name. One challenge I ran into while designing the app was with how the PokeAPI was laid out. Pokemon types were contained in multiple different arrays which lead to having to iterate over them to get the types out and populate them to the page. I found myself researching a lot during this project and learning ways to manipulate JavaScript and JQuery that I didn't learn in the class.

Python

I designed this project for a final in my Python class, what we could make was very open ended but I wanted to design something that utilized a lot of moving parts to really test my programming capabilities. One of the most difficult things I came across while designing this project was storing the tetris pieces currently on the board. I had to use multi-dimensional arrays to store the current board state and iterate over them when completing a line. Python was the first language I learned and it really helped me get a sense of what is required when it comes to programming and it is still a passion to this day.
